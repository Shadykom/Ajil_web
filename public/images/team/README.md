# Team Images

This directory should contain team member photos.

## Required Images

Based on the current code, the following images are expected:

- `chairman.jpg` - Chairman's photo
- `ceo.jpg` - CEO's photo
- `cfo.jpg` - CFO's photo

## Recommended Specifications

- **Format**: JPG or WebP
- **Dimensions**: 400x400 pixels (square, 1:1 aspect ratio)
- **File Size**: Keep under 100KB each
- **Style**: Professional headshots with neutral background

## Note

The leadership section is defined in the code but currently not displayed.
Add the images and enable the leadership section in `/src/app/about/story/page.tsx` when ready.
