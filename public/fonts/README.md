# GE SS Two Font Files

This folder should contain the GE SS Two font files for the AJIL Finance website.

## Required Font Files

Add the following font files to this directory:

### Light (Weight 300) - For body text
- `GESSTwo-Light.woff2`
- `GESSTwo-Light.woff`

### Medium (Weight 500) - For titles
- `GESSTwo-Medium.woff2`
- `GESSTwo-Medium.woff`

### Bold (Weight 700) - For headings and CTAs
- `GESSTwo-Bold.woff2`
- `GESSTwo-Bold.woff`

## Font Information

**GE SS Two** (جي إي إس إس تو) is a modern Arabic corporate font widely used in Saudi banking, fintech, and government websites.

- **Publisher**: Boutros International
- **License**: Commercial license required

## Fallback

If the GE SS Two font files are not available, the website will automatically fall back to Cairo font (loaded from Google Fonts).

## Usage in CSS

The font is configured with the following weights:
- `font-weight: 300` - GE SS Two Light
- `font-weight: 500` - GE SS Two Medium  
- `font-weight: 700` - GE SS Two Bold

In Tailwind CSS, use:
- `font-light` for body text
- `font-medium` for titles
- `font-bold` for strong headings/CTA
