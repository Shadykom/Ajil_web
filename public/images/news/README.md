# News Images

This directory should contain news article images.

## Required Images

Based on the current code, the following images are expected:

- `digital-platform.jpg` - News article about digital platform launch
- `partnership.jpg` - News article about auto dealer partnerships
- `ramadan.jpg` - Ramadan special offers news
- `award.jpg` - Excellence award news
- `branch.jpg` - New branch opening news
- `vision2030.jpg` - Vision 2030 initiatives news

## Recommended Specifications

- **Format**: JPG or WebP for better compression
- **Dimensions**: 800x600 pixels minimum (16:9 or 4:3 aspect ratio)
- **File Size**: Keep under 200KB each for fast loading
- **Quality**: High quality, optimized for web

## Note

The current implementation uses placeholder components (AjilSymbol) when images are missing.
